# K6 Skill Admin Guide

## Responsibilities
- Ensure the skill and templates are up to date
- Review and approve contributions
- Maintain documentation and coverage matrix
- Coordinate with backend and QA teams for contract changes

- Update maintainers list as needed
- Oversee automation and integration plans
- Monitor skill adoption and usage
