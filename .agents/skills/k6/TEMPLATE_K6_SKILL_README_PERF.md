# K6 Skill Performance Guide

## Performance Testing Best Practices
- Use realistic data and traffic patterns
- Model full transaction cycles, not just isolated endpoints
- Monitor system metrics during tests
- Analyze results for bottlenecks and regressions
- Document performance findings in workflow docs

## How to Use
- Follow performance best practices in all k6 scripts
- Update documentation with performance insights
- Collaborate with backend and infra teams for tuning
