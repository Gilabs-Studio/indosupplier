# K6 Skill Automation Plan

## Goal
Automate the process of implementing, documenting, and maintaining k6 transaction workflow tests for GIMS ERP modules.

## Steps
1. **User triggers a skill action** (e.g., new workflow, update docs)
2. **Agent enters Gill Me mode** to collect detailed workflow info
3. **Agent generates/scaffolds code and documentation** using templates
4. **Agent updates coverage matrix and links**
5. **Agent provides recommendations and best practices**
6. **User reviews and iterates as needed**

## Future Automation Ideas
- CLI or UI integration for skill actions
- Automated backend contract validation
- Integration with CI/CD for test and doc sync
- Notification on outdated docs or missing coverage
