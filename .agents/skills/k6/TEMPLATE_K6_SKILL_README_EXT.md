# K6 Skill Extension Guide

## How to Extend the Skill
- Add new templates for additional workflow types or modules
- Update SKILL.md and index.json to register new capabilities
- Document new actions and usage in the user guide
- Collaborate with maintainers for review and approval

## Best Practices
- Follow clean architecture and documentation standards
- Test all new templates and actions before release
