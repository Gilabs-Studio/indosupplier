# K6 Skill Coverage Guide

## Coverage Matrix
- The coverage matrix tracks which modules and workflows are implemented, planned, or blocked
- Update the matrix after every new workflow implementation
- Use the matrix to identify gaps and prioritize future work

## How to Use
- Review the matrix before starting new workflow tests
- Update status and relationships as workflows evolve
- Link matrix updates in documentation index

*Coverage tracking is key to comprehensive and reliable testing.*
