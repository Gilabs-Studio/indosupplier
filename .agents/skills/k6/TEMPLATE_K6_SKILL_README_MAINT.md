# K6 Skill Maintenance Guide

## Maintenance Best Practices
- Regularly review and update all templates and docs
- Sync with backend and business process changes
- Update the coverage matrix and relationship diagrams
- Document all changes in the changelog
- Solicit feedback from users and stakeholders

## How to Use
- Schedule periodic reviews of the skill and documentation
- Assign maintainers for ongoing updates

---
*Ongoing maintenance is key to the long-term value of the k6 skill.*
