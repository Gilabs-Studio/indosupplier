# K6 Skill Audit Guide

## Audit Checklist
- All critical workflows are covered by k6 tests
- Documentation is up to date and matches implementation
- Coverage matrix accurately reflects current state
- Relationship diagrams are current
- Best practices are followed in all scripts and docs

## How to Audit
- Review the coverage matrix and workflow docs
- Check generated scripts for adherence to templates
- Validate that Gill Me sessions are documented
- Report findings to maintainers
