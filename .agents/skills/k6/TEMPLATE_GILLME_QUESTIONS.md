# Gill Me: Workflow Inquiry Questions

Use these questions to drive detailed workflow planning and documentation for any k6 transaction test.

1. **Business Goal**
   - What is the business goal of this workflow?
2. **Main Entities & Documents**
   - Which documents/entities are involved?
3. **Preconditions & Data Setup**
   - What data or preconditions are required?
4. **Workflow Steps**
   - List the main steps in this workflow.
5. **API Mapping**
   - Which API endpoints are involved?
6. **Success & Failure Scenarios**
   - What are the expected success and failure scenarios?
7. **Concurrency & Integration Points**
   - Are there concurrency or integration points to consider?
8. **Clean Architecture Documentation**
   - How should this workflow be documented and linked?

*After each answer, the agent will provide recommendations and best practices.*
