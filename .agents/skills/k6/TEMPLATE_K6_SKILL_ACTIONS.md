# K6 Skill: Supported Actions

## 1. New Workflow Implementation
- Scaffold k6 script and documentation for a new module workflow
- Enter Gill Me mode to collect workflow details
- Generate code and docs using templates

## 2. Update Documentation
- Sync documentation with latest implementation and recommendations
- Update module coverage matrix and links

## 3. Show Coverage Matrix
- Generate or update MODULE_TRANSACTION_COVERAGE.md
- Visualize inter-module relationships and workflow status

## 4. Gill Me Session
- Ask detailed workflow questions
- Provide recommendations after each answer
- Refine workflow plan iteratively

## 5. Template Management
- Provide and update templates for scripts, docs, and inquiry sessions

*Use these actions as commands or workflow triggers for the k6 skill.*
