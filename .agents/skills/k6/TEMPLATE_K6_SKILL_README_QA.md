# K6 Skill QA Guide

## QA Responsibilities
- Validate that all critical workflows are covered by k6 tests
- Review generated scripts and docs for accuracy
- Report gaps or failures in test coverage
- Collaborate with engineering to refine workflows and scenarios

## How to Use
- Use the coverage matrix to identify missing or incomplete tests
- Run k6 scripts to verify workflow correctness
- Update documentation as workflows evolve

---
*QA teams play a key role in maintaining test quality and coverage with this skill.*
