# Feature: Search & Discovery

## Purpose

Search is the primary entry point for buyers on the platform. A buyer who cannot find the right supplier quickly will leave. The search system must handle diverse queries — from precise technical keywords to vague natural language descriptions — and surface the most relevant, trustworthy suppliers in response.

---

## Who Uses This

- **Buyers** — to find suppliers they want to contact
- **Public visitors (no login)** — can search and view results including full supplier contact info; cannot send RFQs

---

## Search Entry Points

1. **Main search bar** on the homepage (most common)
2. **Category page** — buyer browses to a category and filters from there
3. **AI Search toggle** on the main search bar — switches to natural language mode

---

## Standard Search

### How It Works
- Buyer types keywords (product name, supplier name, category name)
- Results show matching suppliers ordered by a relevance score
- Results update as filters are applied

### Filters Available

| Filter | Options |
|---|---|
| Category | All industry categories and subcategories |
| Location | Province (multi-select), then city |
| Verification Level | All / Level 2 and above / Level 3 only |
| Company Type | PT / CV / UD / Cooperative / All |
| Tax Status | All / PKP only (for buyers who need VAT invoices) |
| Certifications | Halal / ISO 9001 / BPOM / SNI / Organic / etc. (multi-select) |
| Export Experience | All / Export-experienced only |
| Language | All / English-speaking suppliers |
| Response Rate | All / "Responsive" only (rate > 80%) |

### Sorting Options
- Relevance (default)
- Highest rated
- Fastest response time
- Newest members
- Most RFQs responded to

### Results Display
- **Card view** (default on mobile): logo, name, city, verification badge, response rate, top 2 categories, star rating
- **List view** (default on desktop): compact, shows more suppliers per screen
- Paid placements appear at the top with clear labels (Auction Placement, Sponsored, Hot, Best)

## AI Search

### Purpose
Allow buyers to describe what they need in natural language — without knowing the exact industry terminology — and let the system figure out which suppliers to show.

### How It Works

1. Buyer enables the **"AI Search"** toggle and types a query in natural language
2. The system sends the query to an AI language model with the instruction to extract specific product keywords and relevant industry categories
3. The AI returns a structured list of keywords
4. Keywords are displayed to the buyer as editable chips / tags — they can remove irrelevant ones or add new ones
5. Buyer confirms and the search runs using those keywords
6. Results are shown grouped by the generated keyword / category

### Example Queries

| Buyer Types | AI Generates |
|---|---|
| "I want to open a new cafe" | espresso machine, coffee grinder, cafe chairs, cafe tables, POS cashier system, food display showcase, cafe kitchen equipment |
| "Uniform fabric for office staff" | cotton fabric, polyester fabric, sewing thread, drill fabric, TC fabric |
| "Eco-friendly food packaging" | paper bag, kraft box, biodegradable packaging, paper food-grade container |
| "Raw materials for local skincare brand" | cream base, emulsifier, preservative, fragrance oil, active skincare ingredients |

### Behavior Rules
- If the query is too short or ambiguous, the AI asks the buyer to describe their need in more detail (with examples shown)
- AI Search results are grouped by category — each group shows the most relevant suppliers
- The buyer can switch back to standard search at any time
- Keywords generated by the AI are shown to the buyer for confirmation before the search runs — the buyer stays in control
- AI Search supports both Bahasa Indonesia and English queries

## Category Pages

Each category (e.g. `/kategori/tekstil`, `/kategori/agrikultur`) has its own dedicated page that:
- Shows all suppliers in that category
- Has a pre-applied category filter
- Is SEO-optimized with a structured URL, meta description, and heading
- Shows Auction Placement slots at the top
- Allows all the same filters as main search

---

## Search Ranking Logic (Organic)

The order of organic results (non-paid) is determined by:

| Signal | Weight |
|---|---|
| Keyword match relevance | High |
| Verification level (Level 2 and 3 rank higher) | Medium |
| Response rate | Medium |
| Profile completeness | Medium |
| Recency of profile update | Low |
| Star rating | Low |

Suppliers with the "Inactive" label (response rate < 10% for 60 days) are excluded from organic results entirely.

---

## Public RFQ Board

A dedicated page (`/rfq-terbuka`) shows active Broadcast RFQs.

- Each entry shows: product category, general description, quantity range, timeline, buyer's country
- Buyer's identity is **not shown** — only their Lead Quality Score badge and country
- Logged-in suppliers can click "I'm Interested" to enter the Broadcast RFQ flow
- Visitors without an account see the board and are prompted to register to respond

This page acts as both a demand signal for potential new suppliers considering joining the platform, and an SEO content surface.
